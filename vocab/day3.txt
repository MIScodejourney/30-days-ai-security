Object Storage

Bucket

Object

Time to Last Byte (TTLB)

Request Duration

Latency

Cross-region
بین دو منطقه جغرافیایی متفاوت

Synthetic Benchmark
آزمایش شبیه‌سازی‌شده برای اندازه‌گیری عملکرد

Workload
بار کاری (میزان درخواست‌ها یا پردازش‌ها)

R2 Gateway Worker

Authentication
احراز هویت

Routing Logic
منطق مسیریابی درخواست‌ها

Durable Object
آبجکت پایدار (مکانیزم ذخیره‌سازی state در Cloudflare)

Metadata

Distributed Storage Infrastructure

Replication

Streaming

Location Hint
راهنمای محل جغرافیایی باکت

Strong Consistency
یکپارچگی قوی (بعد از نوشتن، بلافاصله قابل خواندن است)

Immediately Accessible
